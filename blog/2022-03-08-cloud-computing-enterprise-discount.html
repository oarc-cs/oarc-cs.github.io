Discount programs for commercial clouds#

Discount programs for commercial clouds:

  • AWS Enterprise Discount Program - for discount pricing on the existing UC-wide Amazon Web Services (AWS) Agreement #UCOP-19

  • Google Cloud Platform - request a new GCP account under the existing UC-wide Agreement
